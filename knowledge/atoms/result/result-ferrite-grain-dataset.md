# 核心断言

铁素体晶粒度测试数据集包含来自多个钢材试样的 **82 张金相显微图像**，依据 GB/T 6394—2017 标准，采用 **截点法** 进行晶粒度评级。图像覆盖不同样品位置（边部、心部、四分之一处）和不同放大倍数（200×、500×、1000×）。

## 使用说明

1. **图像质量较好时**：直接拉 3–5 条直线，计算与晶界（黑色线条）的交点数
2. **图像质量较差时**（晶界不明显 / 有析出相）：可进行图像预处理或统计时排除干扰
3. **存在大块第二相时**：若截线与其相交，该交点不计数
4. **测量数据**需包含：交点数 + 相邻两交点之间的距离
