# 文献调研与综述方法论

> 从检索到综述的系统性流程。

---

## 整体流程

```
检索策略 → 初筛 → 精读 → 笔记结构化 → 缺口识别 → 综述写作
```

---

## 第 1 步：检索策略

### 检索数据库

| 数据库 | 覆盖范围 | 适用领域 |
|--------|---------|---------|
| **Web of Science** | 所有学科核心期刊 | 最权威，引用数据全 |
| **Scopus** | 比 WoS 更广的期刊覆盖 | 工程、材料、计算机 |
| **Google Scholar** | 预印本、会议、书籍 | 初探、找全 |
| **PubMed** | 生物医学 | 生命科学 |
| **IEEE Xplore** | 电子工程、计算机 | CS/EE |
| **arXiv** | 预印本 | 最新研究，未审稿 |
| **Semantic Scholar** | AI 增强检索 | 找核心论文 |
| **Connected Papers** | 论文关系图 | 找前驱/后续工作 |

### 构建检索式

```
# 布尔运算符
AND   # 缩小范围：A AND B → 同时包含 A 和 B
OR    # 扩大范围：A OR B → 包含 A 或 B
NOT   # 排除：A NOT B → 包含 A 但不包含 B

# 常用结构
(Topic A) AND (Topic B) AND (Method)
("semantic segmentation" OR "instance segmentation")
AND ("metallography" OR "microstructure")
AND ("deep learning" OR "convolutional neural network")

# 限定
Year: 2020-2025
Type: Article OR Review
Language: English
```

### 检索检查清单

- [ ] 拆解研究问题为 2–4 个核心概念
- [ ] 每个概念列出同义词和近义词
- [ ] 用布尔运算符组合
- [ ] 至少检索 2 个数据库
- [ ] 记录检索式（便于重复和引用）

## 第 2 步：筛选与管理

### PRISMA 筛选流程

```
检索获得 (n)     ← 数据库 + 其他来源
  ↓
去重后 (n)
  ↓
标题/摘要筛选 (n)  ← 按纳入/排除标准
  ↓
全文筛选 (n)       ← 按纳入/排除标准
  ↓
纳入综述 (n)
```

### 纳入/排除标准示例

| 纳入标准 | 排除标准 |
|---------|---------|
| 发表于 2020–2025 年 | 非英文文献 |
| 同行评议期刊/会议 | 学位论文、书籍章节 |
| 使用深度学习方法 | 仅用传统图像处理 |
| 针对金相图像分析 | 针对医学图像 |
| 报告了量化指标 | 没有实验验证 |

## 第 3 步：阅读与笔记

### 三遍读文法

| 遍数 | 目标 | 时间 | 读什么 |
|------|------|------|--------|
| **第 1 遍** | 判断是否相关（5 min） | ~5 min | Title + Abstract + 图表 |
| **第 2 遍** | 理解主要内容（1 hr） | ~1 hr | Intro → Method → Results → Conclusion |
| **第 3 遍** | 批判性评估（2 hr+） | ~2 hr | 仔细通读 + 做笔记 |

### 笔记模板（每篇论文一条）

```markdown
## [作者, 年份] 简短标题

**核心贡献：** 一句话总结

**方法：**
- 关键思路
- 实验设置

**结果：**
- 核心数字/发现

**与本文关系：**
- 可借鉴：...
- 不足/缺口：...
- 可作为基线：...

**关键引用：** [文中提到的核心参考文献]
```

### 阅读策略

- **先读综述**，快速掌握领域全貌
- **引用追踪**：某篇论文的引用 → 找后续发展
- **作者追踪**：关注核心课题组
- **反向追踪**：参考文献 → 找源头方法
- **批量读**：同一主题的论文集中读，便于对比

## 第 4 步：研究缺口识别

### 六种典型缺口

| 缺口类型 | 特征 | 示例表述 |
|---------|------|---------|
| **证据缺口** | 某问题尚无实验证据 | "尚未在动态流数据上测试" |
| **方法缺口** | 现有方法不适用 | "传统方法无法处理不平衡数据" |
| **人口/样本缺口** | 只研究了某类样本 | "仅研究了单一钢材牌号" |
| **理论缺口** | 现有理论无法解释 | "现有模型无法解释这种非线性行为" |
| **实践缺口** | 理论与应用间有差距 | "方法虽好但计算成本过高" |
| **元分析缺口** | 缺乏系统性的综合分析 | "目前缺乏大规模的跨数据集对比研究" |

### 缺口发现技巧

1. **限制条件法**：论文中"但"、"however"、"limitation" 后面的内容
2. **对比法**：同一问题的不同方法之间是否有不一致的结论
3. **迁移法**：其他领域的成熟方法是否可用于本领域
4. **时间线法**：某个方向的研究是否近年停滞了

## 第 5 步：综述写作

### 综述论文结构

| 章节 | 内容 |
|------|------|
| **Abstract** | 综述范围、主要发现、结论 |
| **1. Introduction** | 领域背景、为什么需要这个综述、综述范围 |
| **2. 背景介绍** | 核心概念、术语定义、分类体系 |
| **3. 方法分类** | 按方法/主题分组讨论，每组含：代表性工作 → 优缺点 → 演进 |
| **4. 对比分析** | 表格对比各方法在相同指标上的表现 |
| **5. 现有挑战** | 当前方法共同的局限性和未解决的问题 |
| **6. 未来方向** | 研究缺口 → 可能的技术路线 |
| **7. Conclusion** | 总结 |

### 文献分组策略

```
分组标准:
  按方法类型（CNN-based / Transformer-based / GAN-based）
  按任务（分割 / 检测 / 分类）
  按时间线（早期 / 近期 / 最前沿）
  按性能水平（SOTA / 次优 / 基线）

每组讨论格式:
  1. 代表工作的核心思路
  2. 与上一组/上一代的区别
  3. 优点 + 局限
  4. 演进趋势
```

### 引用策略

- **关键奠基工作**必须引
- **最新 SOTA**必须引（近 2 年）
- **对比方法**必须引
- 避免过度自引
- 综述引用 ≥ 100 篇
