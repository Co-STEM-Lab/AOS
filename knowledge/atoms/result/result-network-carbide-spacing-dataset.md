# 核心断言

网状碳化物+珠光体片层间距智能识别数据集包含 **网碳** 和 **珠光体片层间距** 两个子集，用于钢材金相图像中网状碳化物识别和珠光体片层间距测量的 AI 模型训练与评测。标注规范由数据集内的 `网状碳化物和珠光体片层间间距.md` 文档定义。

## 数据来源

原始金相图像来自首钢驻场检测资料，经人工标注测量后整理。标注规范文档 `网状碳化物和珠光体片层间间距.md`（位于数据集目录内）定义了红色/蓝色标记规则及测量需求。

## 标注规范（来自 `网状碳化物和珠光体片层间间距.md`）

### 网状碳化物识别
- **红色线**：标记网状碳化物，其环绕的区域为 1 个晶粒
- **蓝色线**：标记晶粒的最大长度、最小长度、等效圆直径
- **蓝色线**：标记网状碳化物的厚度

### 珠光体片层间距测量
- **红色线**：标记 11 个珠光体片层
- **蓝色（谷）**：标记片层间的 10 个间距
- 若红色线长度为 1mm，则间距 = 1mm / 10 = 0.1mm

## 数据集组成

### 子集一：网碳（network-carbide/）

| 组件 | 数量 | 格式 | 说明 |
|------|------|------|------|
| 小尺寸标注图 | 3 | RGBA PNG | 442×332，疑似截图或示例 |
| 中尺寸标注图 | 9 | RGB PNG | 649×487，含标注的微视图 |
| 大尺寸原始图 | 7 | Palette PNG | 2048×1536 或 1024×768，带测量标注 |
| 大尺寸 TIF 原图 | 7 | TIF | 与上对应的原始 TIFF 文件 |

### 子集二：珠光体片层间距（pearlite-spacing/）

| 组件 | 数量 | 格式 | 说明 |
|------|------|------|------|
| 标注图 | 65 | Palette PNG | 1024×768，含测量标注 |
| 原始 TIF | 65 | TIF | 1024×768，未加工原图 |

### 图像命名

珠光体片层间距命名格式：

```
{参数A}-{参数B}-{序号}.{ext}
例：0.25-0.5-01.png → 参数A=0.25, 参数B=0.5, 序号01
```

> ⚠️ 参数 A/B 的具体含义（碳含量？腐蚀浓度？放大倍数？）未确认。参数 B 取值：0, 0.25, 0.5；参数 A 取值多样（0.25, 0.5, 4, 64~100, 280, 281 等）。

网碳命名格式：

```
{序号}.{ext}
例：13.png → 序号 13
```
