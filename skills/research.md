# 研究方法论

> 从问题到结论的完整研究流程规范。

---

## 研究流程

```
问题定义 → 文献调研 → 假设提出 → 实验设计 → 数据收集
→ 分析验证 → 结论提炼 → 论文写作
```

---

## 第 1 步：问题定义

### 好问题的四个特征

| 特征 | 解释 | 反面例子 |
|------|------|---------|
| **具体** | 有明确的边界条件 | "提高性能"（太泛） |
| **可验证** | 有可测量的指标 | "看起来更好"（不可测） |
| **有边界** | 知道什么情况下不适用 | "适用于所有材料"（太宽） |
| **有动机** | 回答了为什么这个重要 | "因为没人做过"（理由不足） |

### 问题模板

```
在 [场景/材料/条件] 下，[方法/模型] 能否 [达到什么效果]？
相比于 [基线方法]，[本文方法] 的 [指标] 是否能提升 [多少]？
```

**示例：**
> 在钢材金相图像中，基于深度学习的语义分割方法能否自动区分碳化物析出相和片层珠光体？
> 相比于传统阈值分割，本文方法的 mIoU 是否能提升至少 15%？

### 检查清单

- [ ] 这个问题有确定的答案吗？（是/否 或 可量化数值）
- [ ] 同行认为这个问题值得做吗？
- [ ] 有可对比的基线方法吗？
- [ ] 你有数据/设备/手段来回答它吗？

## 第 2 步：假设提出

### 假设格式

**零假设 H₀：** 变量 X 对结果 Y 无显著影响。
**备择假设 H₁：** 变量 X 对结果 Y 有显著影响（或：正向影响）。

### 示例

```
H₀: 使用 AI 分割与人工标注在 Dice 系数上无显著差异。
H₁: 使用 AI 分割的 Dice 系数显著高于人工标注（p < 0.05）。
```

### 假设来源

- 文献中的矛盾或空白
- 已有理论的延伸
- 对实验现象的观察
- 与常识相悖的猜测

## 第 3 步：实验设计

### 核心要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **对照组** | 不做处理 / 用现有方法 | 传统图像分割方法 |
| **实验组** | 施加处理 | 本文的深度学习分割方法 |
| **变量** | 自变量（你改什么）+ 因变量（你测什么） | 方法 → 分割精度、推理速度 |
| **控制变量** | 保持不变的条件 | 同一数据集、同一硬件 |
| **重复** | 每组至少 3 次 | 5 折交叉验证 |
| **随机化** | 样本分配不能有偏 | 随机划分训练/测试集 |

### 常见实验类型

| 类型 | 用途 | 特点 |
|------|------|------|
| **对比实验** | 本文方法 vs 基线 | 最常用，需控制变量一致 |
| **消融实验** | 验证某个模块的必要性 | 逐步去掉组件看性能变化 |
| **参数敏感性** | 找最优超参数 | 扫描一个参数，其他固定 |
| **泛化性** | 跨数据/跨领域验证 | 在不同数据集上测试 |
| **统计检验** | 差异是否显著 | t 检验、ANOVA、Mann-Whitney |

### 实验设计检查清单

- [ ] 对照组是否合理？
- [ ] 基线方法是否已复现并调优？
- [ ] 样本量是否足够（功效分析）？
- [ ] 是否做了随机化？
- [ ] 是否有潜在的混淆变量？
- [ ] 实验是否可重复？

## 第 4 步：数据收集与管理

### 数据管理规范

```
paper-name/
├── data/
│   ├── raw/          # 原始数据（只读，不改动）
│   ├── processed/    # 清洗/预处理后数据
│   └── results/      # 实验结果输出（CSV/JSON）
├── scripts/
│   ├── preprocess.py  # 数据预处理
│   ├── train.py       # 训练
│   └── evaluate.py    # 评估
└── figures/
    └── *.png/pdf      # 出图脚本与输出一致
```

### 数据记录要求

- 每个实验记录：**日期、参数、结果、环境**（见 `scripts/` 中自包含脚本）
- 随机种子固定，保证可复现
- 所有数据有版本（直接用 Git）

## 第 5 步：结果分析与统计检验

### 统计方法选择

| 场景 | 方法 | 适用条件 |
|------|------|---------|
| 两组比较 | Student's t-test | 正态分布、方差齐 |
| 两组比较 | Mann-Whitney U | 非正态分布 |
| 多组比较 | ANOVA + Tukey HSD | 正态分布、方差齐 |
| 多组比较 | Kruskal-Wallis | 非正态分布 |
| 配对比较 | Paired t-test | 同一组前后对比 |
| 相关性 | Pearson / Spearman | 线性 / 单调关系 |
| 分类指标 | Chi-square test | 计数数据 |

### 结果报告规范

- 报告均值 ± 标准差：`85.3 ± 2.1%`
- 报告样本量：`n = 20`
- 报告效应量：Cohen's d / η²
- 统计显著性：`p = 0.003`（而非 `p < 0.05`）
- 置信区间：`95% CI [82.1, 88.5]`

### 可视化对应

| 数据类型 | 推荐图表 | 避坑 |
|---------|---------|------|
| 分类对比 | 柱状图 + 散点 + 误差线 | 只用柱状图不显示分布 |
| 连续趋势 | 折线图 + 置信带 | 点太多连看不清 |
| 分布 | 箱线图 / 小提琴图 | 只用均值不显示分布 |
| 相关性 | 散点图 + 回归线 | 只报相关系数不给图 |
| 高维数据 | t-SNE / UMAP | 不解释参数 |

## 第 7 步：结论提炼

### 结论要素

1. **总结发现** — 用一句话回答研究问题
2. **量化支撑** — 带数字的证据
3. **边界条件** — 在什么条件下有效
4. **意外发现** — 实验中观察到的非预期现象
5. **局限性** — 方法/数据/理论的限制

### 结论与 Discussion 的区别

| Discussion | Conclusion |
|-----------|-----------|
| 解读结果的含义 | 回答研究问题 |
| 与已有工作对比 | 提炼核心贡献 |
| 讨论机制/原因 | 明确边界条件 |
| 展开理论意义 | 指出未来方向（≤2 条） |
