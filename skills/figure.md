# 学术图片规范

> 主流期刊（Nature、Cell、Elsevier、IEEE、ACS、RSC）的图片初版要求。
> 投稿前按此清单检查，避免因格式问题被退回。

---

## 通用参数速查

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 分辨率 | 300 DPI（彩色）<br>600 DPI（灰度）<br>1200 DPI（线条图） | Nature 要求 ≥300 DPI<br>Elsevier 要求 ≥300 DPI |
| 字体 | **Arial** 或 **Helvetica** | 绝大多数期刊推荐无衬线字体 |
| 字号 | 6–8 pt（坐标轴标签）<br>8–10 pt（panel 标签/图例）<br>10–12 pt（图题） | 太小不可读，太大约稿方会缩小 |
| 线宽 | 0.5–2 pt | 坐标轴线 0.5 pt，数据线 1–2 pt |
| 颜色模式 | **RGB**（在线投稿）<br>**CMYK**（印刷） | 初稿一律用 RGB，录用后出版社转 CMYK |
| 文件格式 | **PDF** / **EPS**（矢量）<br>**TIFF**（位图·印刷）<br>**PNG**（位图·屏幕） | Nature 接受 AI/PDF/EPS/TIFF<br>Elsevier 接受 EPS/PDF/TIFF/PNG |
| 宽度 | 85 mm（单栏）<br>170 mm（双栏）<br>140 mm（1.5 栏） | 投稿前确认目标期刊栏宽 |
| 色彩空间 | sRGB | 最通用的色彩空间，屏幕显示与印刷接近 |

## Panel 排版规范

### 多图组合

```
┌─────┬─────┐
│  A  │  B  │
│     │     │
├─────┼─────┤
│  C  │  D  │
│     │     │
└─────┴─────┘
```

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **panel 间距** | 2–5 mm（约 0.1–0.2 inch） | 太挤：标签被遮挡；太空：浪费版面 |
| **panel 标签** | 大写字母 **A B C D** | 放在每个 panel 的**左上角**（或左下角空白处） |
| **标签字体** | 8–10 pt，**粗体** | 与图内正文文字区分 |
| **对齐方式** | 同一行各 panel 顶部对齐 | 使用排版软件的 align 工具 |
| **边框** | 所有 panel 有统一粗细的边框 | 有数据的图（电镜图、照片）必须有 scale bar |
| **scale bar** | 左下角或右下角 | 包含长度标注，如 "2 µm" |

### Panel 排列原则

1. **阅读顺序** — 从左到右、从上到下（A→B→C→D）
2. **逻辑层级** — 方法→结果（不反）
3. **相同比例** — 同一行或同一列的图片保持相同的放大倍数
4. **空白最小化** — 裁剪图片边缘的无效区域

### 常用排版软件

| 软件 | 适用场景 | 学习成本 |
|------|---------|---------|
| **Illustrator** | 专业矢量排版，最推荐 | 中 |
| **Inkscape**（免费） | 功能接近 AI，开源 | 中 |
| **Python (matplotlib)** | 数据图直接排版 | 低（脚本化） |
| **PowerPoint/Keynote** | 简单组图，快速出图 | 低 |
| **Affinity Designer** | Illustrator 替代，一次性付费 | 中 |

## 配色方案

### 通用原则

- 不超过 **6 种** 颜色在同一张图中
- 使用**色盲友好**调色板（~8% 男性是色盲）
- 避免红-绿对比（最常见的色盲类型）
- 用**形状 + 颜色**双重编码（如圆形红色、三角蓝色）

### 推荐调色板

#### 1. Nature 风格

```
主色: #0072B2 (蓝)   #D55E00 (橙)   #009E73 (绿)
辅色: #F0E442 (黄)   #56B4E9 (浅蓝) #CC79A7 (粉)
```

#### 2. Elsevier 风格

```
主色: #00468B (深蓝)  #ED0000 (红)   #0099B4 (青)
辅色: #42B540 (绿)   #FFE200 (黄)   #7C2E9B (紫)
```

#### 3. 灰度友好（适合打印）

```
顺序: #000000 → #555555 → #AAAAAA → #CCCCCC
区分: 不同灰阶 + 不同线型（实线/虚线/点线）
```

#### 4. Color Universal Design（色盲友好）

```
#0072B2  #E69F00  #009E73  #F0E442  #56B4E9  #D55E00  #CC79A7
```

取色工具：https://colorbrewer2.org

### 配色检查清单

- [ ] 灰度打印后是否仍可区分？
- [ ] 红绿色盲（deuteranopia）下是否可读？
- [ ] 颜色有语义含义吗？（如红色 = 实验组，蓝色 = 对照组）
- [ ] 图例与图中颜色一一对应？

## 字体规范

### 推荐字体

| 用途 | 字体 | 备选 |
|------|------|------|
| 坐标轴标签 | **Arial** / **Helvetica** | Myriad Pro, Noto Sans |
| Panel 标签 | **Arial Bold** | Helvetica Bold |
| 图例文字 | **Arial** | 同上 |
| 数据标注 | **Arial** | 同上 |
| 数学符号 | **Times New Roman / Symbol** | 遵循期刊模板 |
| 中文标注 | **Noto Sans CJK** / **Microsoft YaHei** | 中文期刊 |

### 字号层级

```
图题:          10–12 pt (在图片下方)
Panel 标签:     8–10 pt Bold
坐标轴标题:     8–9 pt
坐标轴刻度:     7–8 pt
图例:           7–8 pt
数据标注:       6–7 pt
脚注/星号:      6–7 pt
```

### 字体原则

- 同一论文所有图片使用**同一套字体和字号**
- 坐标轴标题不用斜体（变量符号用斜体）
- 单位之间空一格：`5.2 mm`（非 `5.2mm`）
- Panel 标签在**所有 panel 中保持一致的位置和大小**

## 常用图片类型规范

### 数据图（matplotlib / Excel 生成）

```python
import matplotlib.pyplot as plt

plt.rcParams.update({
    'font.family': 'sans-serif',
    'font.sans-serif': ['Arial'],
    'font.size': 8,
    'axes.titlesize': 9,
    'axes.labelsize': 8,
    'xtick.labelsize': 7,
    'ytick.labelsize': 7,
    'legend.fontsize': 7,
    'figure.dpi': 300,
    'savefig.dpi': 300,
    'savefig.bbox': 'tight',
    'lines.linewidth': 1.5,
    'axes.linewidth': 0.5,
})
```

| 要求 | 规范 |
|------|------|
| 坐标轴 | 有标签和单位，如 `Energy (eV)` |
| 误差线 | 必须标注（SD、SEM 或 CI） |
| 刻度方向 | 朝内或朝外，整本论文统一 |
| 图例 | 框线 ≤ 0.5 pt，背景白色 |
| 统计标注 | `*p < 0.05` `**p < 0.01` `***p < 0.001` |

### 示意图（Illustrator / Inkscape 绘制）

| 要求 | 规范 |
|------|------|
| 线宽 | 统一（建议 1–2 pt） |
| 填充色 | 半透明优先，便于层叠 |
| 标注 | 与图中元素有足够留白 |
| 箭头 | 箭头大小统一，末端刚好接触到标注对象 |

### 电镜图（TEM/SEM 照片）

| 要求 | 规范 |
|------|------|
| Scale bar | **每张图必须有**，放在左下角或右下角 |
| 标注 | 标注关键特征（如用箭头 + `→` 标注） |
| 裁剪 | 去除无效边缘，保持主体居中 |
| 对比度 | 同一组图的亮度和对比度一致 |

## 期刊具体要求索引

| 期刊 | 文件格式 | DPI | 色彩 | 特殊要求 |
|------|---------|-----|------|---------|
| **Nature** | AI / EPS / PDF / TIFF | ≥ 300 | RGB | 每张图单独文件；图题在正文末尾 |
| **Science** | EPS / PDF / TIFF | ≥ 300 | CMYK | 图片宽度 ≤ 8.5 inch |
| **Elsevier** | EPS / PDF / TIFF / PNG | ≥ 300 | RGB | 通过 EES 系统上传，每张图单独文件 |
| **IEEE** | EPS / PS / TIFF / PNG | ≥ 600 | RGB | 不允许嵌入图中文字（文字在图题中） |
| **ACS** | EPS / PDF / TIFF | ≥ 300 | RGB | 图形摘要需单独准备（5×3 inch） |
| **RSC** | EPS / PDF / TIFF / PNG | ≥ 300 | RGB | 彩色图免收彩版费 |
| **PNAS** | EPS / PDF / TIFF | ≥ 300 | CMYK | 宽 ≤ 8.7 cm（单栏）/ ≤ 17.8 cm（双栏） |
